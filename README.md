# HowLongDay

SEO-driven programmatic site showing sunrise, sunset, golden hour, blue hour
and daylight duration for 50,000+ cities worldwide.

See [`CLAUDE_1.md`](./CLAUDE_1.md) for the full product spec, brand system and
dynamic-sky design.

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Vercel hosting
- Open-Meteo Astronomy API + SunCalc

## Dev
```bash
npm install
npm run dev
```

## Status
v0 — branded landing page. Programmatic city pages, sky system and search
land incrementally on top of this scaffold.
